## ✨ Overview

The **CAL Legal Chatbot** is a privacy-first legal triage platform that helps Illinois residents quickly understand their legal situation and connect with the right resources.

Instead of navigating complex legal systems, users can:
- Answer guided questions  
- OR describe their issue in plain English  

…and the system intelligently guides them to appropriate legal help.

---

## 🧠 AI-Powered Assistance

This system includes a lightweight AI layer that understands **natural language input**.

💬 Example:
> “I can’t get into my apartment”

➡️ Automatically classified as a **housing issue**  
➡️ Instantly provides relevant legal resources  

This removes friction for users who don’t know legal terminology.

---

## ⚡ Key Features

- 🧭 Guided legal triage (housing, child support, education, etc.)  
- 🤖 AI-based issue detection from user input  
- 🚨 Emergency handling (*Yes / No / I don’t know*)  
- 📞 Resource recommendations with links + phone numbers  
- 🔁 Restartable chatbot flow  
- 🔊 Accessibility: text-to-speech (read aloud)  
- 🔒 Privacy-first design with discreet mode concept  
- 📱 Fully responsive (mobile-friendly)  

---

## 🏛 Legal Resources Included

Users are connected to trusted organizations such as:

- **Chicago Advocate Legal, NFP**  
- **Greater Chicago Legal Clinic**  
- **CARPLS**  
- **Justice Entrepreneurs Project**  

Each resource includes:
- Direct website links  
- Intake phone numbers  
- Relevant service information  

Additional coverage:
- Housing assistance  
- Family law (divorce, custody, child support)  
- Education law (including PDF resources)

## ⚙️ Local Development

### 🔹 Backend

```bash
cd backend
python -m venv venv

# Windows
venv\Scripts\activate

# Mac/Linux
source venv/bin/activate

pip install -r requirements.txt
uvicorn main:app --reload
```
###🔹 Frontend
```bash
cd frontend
npm install
npm run dev
```
👉 Open: http://localhost:5173
